# CST8318 - Graphic Technologies

# Course Introduction

Students are introduced to graphics and illustration programs in the computer environment. Students also learn skills and production techniques which help to design effective interfaces for websites, mobile websites and mobile applications. Focus is placed on hands-on learning of software, such as Adobe Photoshop and Illustrator. Basic design concepts are covered within the context of becoming productive with the software packages.

# Technology Stack

# Core Topics

  1. Knowledge with these concepts in Adobe Photoshop:
  • Organize workflows
  • Selecting and cutting a silhouette
  • Clean up a silhouettes edges
  • Correcting colors for vibrancy
  • Perfecting pixels using special tools
  • Blend layers to create a single coherent image
  • Save file formats for the world wide web
  1. Knowledge of these following concepts in Adobe Illustrator:
  • Use the software packages to create icons
  • Use colour effectively in designing professional icons
  • Use distinct shapes to create professional icons
  • Use allignment and proper typography
  • Export icon files in a variety of sizes
  1. Scalable Vector Graphics (SVG) and SVG markups
  • linking SVG buttons with states (on, off, not connected)
  • creating HTML/CSS to publish buttons on various mobile devices
  1. User interface interaction design concepts
  • Putting it all together
