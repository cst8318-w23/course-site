# Exercise 6 — Animated SVG

::: Tip Files Starter files (opens new window) :::

# Instructions

Create an application loader using an animated SVG. To complete this exercise you must:

  • Create your loader in Illustrator and prep the file for export via the Export as... dialogue.
  • Copy the SVG code into the index.html file.
  • Create the animation in the main.css file.

Your SVG code should be inside of the index.html file and the styling should all be written in the ./css/main.css file.

# Example

# File Structure

Submit a single compressed folder title fName-lName-section#-animatted-svg containing the following:

  • ./css
    • main.css
  • index.html
  • Loader.ai
